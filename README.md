## Getting Started

### Prerequisites

- Node version ^18.3.0

  ```sh
  https://nodejs.org/en/
  ```

- npm ^8.11.0

  ```sh
  https://www.npmjs.com/
  ```

### Setup

1. Clone the repo

   ```sh
   git clone -b main --single-branch https://github.com/KOFI-GYIMAH/mern-admin-client.git
   ```

2. Install Packages

   ```sh
   npm install
   ```

3. Compile and Hot-Reload for Development

   ```sh
   npm run dev
   ```

4. Compile and Minify for Production

   ```sh
   npm run build
   ```

5. Locally preview production build

   ```sh
   npm run start
   ```
